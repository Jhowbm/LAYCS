═══════════════════════════════════════════════════════════════════
  OPÇÃO 1: USAR O ARQUIVO .BAT (RECOMENDADO PARA WINDOWS)
═══════════════════════════════════════════════════════════════════

1. Vá até a pasta do projeto:
   C:\Users\Bispo\.Projetos\flashscore-lay

2. Dê um duplo clique no arquivo:
   INICIAR.bat

3. Aguarde o processo automático:
   ✓ Verificar ambiente virtual
   ✓ Instalar dependências
   ✓ Iniciar aplicação

4. Acesse no navegador:
   http://localhost:5000

5. Na interface, clique na aba "Método de Ciclos"


═══════════════════════════════════════════════════════════════════
  OPÇÃO 2: CRIAR ATALHO NA ÁREA DE TRABALHO
═══════════════════════════════════════════════════════════════════

1. Vá até a pasta do projeto

2. Dê um duplo clique no arquivo:
   CRIAR_ATALHO.bat

3. Um atalho será criado na sua área de trabalho:
   "Flashscore Lay Analyzer.lnk"

4. Depois basta dar um duplo clique no atalho na área de trabalho!

═══════════════════════════════════════════════════════════════════
  PRIMEIRO USO - MÉTODO DE CICLOS:
═══════════════════════════════════════════════════════════════════

1. Acesse: http://localhost:5000

2. Clique na aba: "Método de Ciclos"

3. Leia o aviso sobre operação em dinheiro real

4. Clique em "Criar Ciclo"

5. Escolha:
   - Perfil: Conservador (4%), Moderado (4-5%) ou Agressivo (8%)
   - Banca Inicial: ex: R$ 100

6. Clique em "Criar Ciclo"

7. Adicione suas entradas com os dados dos jogos

═══════════════════════════════════════════════════════════════════

No terminal onde o sistema está rodando:
- Pressione: CTRL + C

Ou simplesmente feche a janela do terminal.


═══════════════════════════════════════════════════════════════════
  SOLUÇÃO DE PROBLEMAS:
═══════════════════════════════════════════════════════════════════

❌ O navegador não abre:
   - Abra manualmente: http://localhost:5000
   - Verifique se a porta 5000 não está em uso

❌ Erro de permissão:
   - Execute como Administrador
   - Verifique permissões da pasta

❌ Erro de dependências:
   - O sistema instala automaticamente
   - Se falhar, execute: pip install flask selenium beautifulsoup4

❌ Porta 5000 ocupada:
   - Edite o arquivo app.py (final do arquivo)
   - Mude port=5000 para port=5001
   - Acesse: http://localhost:5001

═══════════════════════════════════════════════════════════════════
  ACESSO REMOTO - COMUNICAÇÃO LOCAL ↔ NUVEM
═══════════════════════════════════════════════════════════════════

🌐 URL NA NUVEM: https://laycs.onrender.com

Para usar a análise de jogos através da nuvem (de qualquer lugar):

1. Execute INICIAR.bat (sistema local deve estar rodando)
2. Configure acesso remoto:
   - Execute ACESSO_REMOTO_NGROK.bat (recomendado)
   - Ou configure IP público (veja CONFIGURAR_IP_NUVEM.md)
3. Configure a variável LOCAL_API_URL no Render
4. Acesse https://laycs.onrender.com e use a aba "Análise"

📋 Documentação completa: COMUNICACAO_LOCAL_NUVEM.md

⚠️  O Método de Ciclos funciona 100% na nuvem sem necessidade de
   configuração adicional. Use a aba "Método de Ciclos" diretamente
   em https://laycs.onrender.com
